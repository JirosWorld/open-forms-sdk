# Buttons

## Variants

The Open Forms button wraps the Utrecht Button to deal with the disabled state. Currently, the
Utrecht button sets the 'disabled attribute' on the button, but this means that it is not possible
to tab navigate to it and screen readers will skip it. With our wrapper we set the 'aria-disabled'
aria attribute instead.

More information on the Utrecht buttons can be found
[here](https://nl-design-system.github.io/themes/?path=/docs/button--gemeente-utrecht).
