# Anchors

Anchors are used for clickable links. Optionally, you can hook into the `click` event
with the `onClick` prop, which will fire in addition to the default click handler.

The `Anchor` components wraps around the `UtrechtLink`
[component](https://nl-design-system.github.io/utrecht/storybook/?path=/docs/css-link-readme--page)
and therefore supports the relevant `--utrecht-link-*` design tokens. Additional
Open Forms specific styles are added via the `--openforms-*` modifiers.

This component can and should be used whenever using `react-router` `LinkAnchor`
components, as it implements the required handlers which `UtrechtLink` doesn't.

export const Template = ({label, ...args}) => <Anchor href="https://example.com" {...args}>{label}</Anchor>;

## Available design tokens

Anchor components can be styled through design tokens. See the
[upstream](https://nl-design-system.github.io/utrecht/storybook/?path=/docs/css-link-tokens--page)
documentation for the available `.utrecht-link` tokens.

Below are Open Forms' extensions to the existing tokens in CSS-variable format:

```css
.openforms-theme {
  /*
    falls back to --of-typography-sans-serif-font-family or
    --utrecht-document-font-family
  */
  --of-utrecht-link-font-family: <font-family>;
  /* falls back to --of-color-fg-muted */
  --of-utrecht-link-muted-color: <color>;
}
```
