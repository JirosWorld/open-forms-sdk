# FormikDecorator

Use the `FormikDecorator` to wrap a (form field) component in a `<Formik>` container, providing the
necessary context.

Supported parameters are :

- `initialValues`: a mapping of field name (string) to field value (any)
- `initialErrors`: a mapping of field name (string) to field error message (string)

Field names may contain periods (`.`) and square brackets (`[42]`) for lodash-style lookups, see the
Formik documentation.

## Example

```jsx
import {Field} from 'formik';

import {FormikDecorator} from 'story-utils/decorators';

export const Default = {
  render: () => <Field type="text" as="input" name="test" />,
  decorators: [FormikDecorator],
  parameters: {
    formik: {
      initialValues: {test: 'some value'},
      initialErrors: {test: 'an error'},
    },
  },
};
```
