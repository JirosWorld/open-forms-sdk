# Field Set

A fieldset is used to group a number of fields. The provided label will be used as title of the
fieldset. Optionally, you can hide this title.

A fieldset is a presentation-only component - no data is saved to the API.

Additionally the use of a field set increases accessibility: grouping related form controls makes
forms more understandable for all users, as related controls are easier to identify, making it
easier for people to focus on smaller and more manageable groups.

**Legend**

A Field Set label that will appear as a Legend for the component in the form. The provided component
label goes in the legend.
